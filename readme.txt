회원관리 및 게시판 기능.
회원관리, 회원가입, 정보변경, 탈퇴 등.
게시판 무제한 생성.
글을 읽고 쓸 수 있으며 간단한 답글을 남길 수 있는 게시판.

PHP 코드 설명.
PHP 코드 소스는 개인적으로 사용시 수정하여 사용할수 있지만 수정된 프로그램의 재배포는 금지.
